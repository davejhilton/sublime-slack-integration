====================
Thank you for installing Sublime Slack Integration plugin.
===================

With this plugin you can:
    * Sends selected code to slack (multiple cursors == multiple messages)
    * Send custom message to slack.com
    * choose channel when sending messages
    * Multiple teams support

=========== Usage ===========
* ctrl+alt+u -> Send Selection
* ctrl+alt+n -> Send Custom Message

=========== TO DO ===========
Please set your API "token" in

    Menu: Preferences
    -> Package Settings
    -> Slack
    -> Settings - User


Example:

{
    "team_tokens": {
        "Team 1": "team-1-token-goes-here",
        "Team 2": "team-2-token-goes-here",
    }
}

You can get the token from https://api.slack.com/#auth (make sure you are authenticated on slack.com with your account)
